    What is used to pass data between React components?

    1 / 1 point
    Correct

    Correct. Props, short for properties, are used to pass data between React components. Props are read-only because Reacts data flow between components is uni-directional, from parent to child only.

    Question 4

    When does a component lifecycle start?

    1 / 1 point
    Correct

    Correct. A component lifecycle starts when components are created or mounted on the DOM. Once mounted, they grow by updating and end by being unmounted from the DOM.

    Correct

    Which function is invoked right after the component is mounted on the DOM?

    1 / 1 point
    Correct

    Correct. The componentDidMount() function is invoked right after the component is mounted on the DOM or, in other words, after the render() function is executed for the first time.

    What do you use to make a call to an external server asynchronous?

    1 / 1 point
    Correct

    Correct. Most requests to external servers are blocking. To make the call asynchronous, you can use promises.
